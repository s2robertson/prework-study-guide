HTML

  • The head element contains information about the webpage.
    • The title element defines the title for a webpage.
    • The meta element defines metadata such as the character set, description, author, and viewport.
    • The style element defines CSS styles for a webpage.
    • The script element embeds scripts (i.e. dynamic behavior) in the HTML document.
    • The link element connects external resources (e.g. stylesheets, scripts) to the HTML document.
  • The body element represents the visible content to the user.
    • The h1, h2, ... h6 elements represent headings.
    • The p element represents a paragraph.
    • The ol and ul elements represent ordered and unordered lists, and the li element represents a list item.
    • The a element represents a link (also known as an anchor) to another page.
    • The img element represents an image.
    • The header, footer, main, and section elements represent semantic divisions of the page content.

Git

  • git status: checks which branch we are currently on
  • git add [file]: stages a file for commit
  • git reset [file]: unstages a file
  • git commit -m "[message]": commits staged changes
  • git diff: compares the working tree against the index
  • git diff --staged: compares the index against the last commit
  • git branch [branch name]: creates a new branch
  • git checkout [branch name]: switches to a branch
  • git checkout -b [branch name]: creates a new branch and switches to it
